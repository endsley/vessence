What is Jane?

Jane is your personal AI assistant. She remembers your conversations, files, preferences, and projects through a persistent memory system (ChromaDB). The more you interact with her, the better she understands you.

What are Essences?

Essences are pluggable AI personalities. Each essence gives Jane a new role — a tax accountant, a news briefer, a music DJ. When you activate an essence, Jane loads that role's knowledge, tools, and personality while keeping her core memory of you.

  • Activate: Click an essence on the Essences page. Jane switches to that persona.
  • Deactivate: Click "Back to Jane" in the header to return to default Jane.
  • Memory: Each essence has its own isolated memory. Jane's universal memory is always available.

What are Tools?

Tools are standalone utilities that essences and Jane can invoke to get things done. Examples include the file indexer, vault search, text-to-speech, and memory management tools. Each tool does one specific thing well.

How they appear

Installed tools show up in the Tools dropdown in the header. Jane loads the list at startup by scanning the tools directory.

How they're used

Jane calls tools automatically when your request is relevant — for example, if you ask to search your vault, she invokes the vault search tool behind the scenes. You can also reference a tool by name to be explicit.

How to create your own tool

Create a Python file in the tools/ directory inside your essences folder. Your tool must expose function signatures that Jane can discover. At minimum, define:

  • A clear function name — descriptive of what it does (e.g., search_notes)
  • A docstring — Jane reads this to understand when to call your tool
  • Typed parameters — use Python type hints so Jane knows what arguments to pass
  • A return value — return a string or dict that Jane can relay to the user

Look at existing tools in the essences directory for working examples.

Tools vs Essences

Tools are single functions — they do one job and return a result. Essences are complete personas with their own memory, personality, and multiple tools combined. Think of tools as the building blocks that essences are made from.

Example: A "vault search" tool finds files. The "Life Librarian" essence uses that tool plus memory, personality, and conversation skills to act as a full personal archivist.

Job Queue vs Prompt List

Prompt List

Quick tasks you want Jane to handle when you're away. Type prompt: do X and Jane queues it. When you go idle for 5+ minutes, Jane picks up the next prompt and works through it autonomously.

Good for: bug fixes, file cleanup, small features, research tasks.

Job Queue

Bigger, multi-step projects with full context. Each job is a detailed spec file with objective, steps, verification criteria, and files involved. Create jobs with add job: and execute with run job queue:.

Good for: building features, refactoring, audits, Docker builds, anything that needs a plan.

When to use which: If you can describe it in one sentence → prompt list. If it needs a plan with multiple steps → job queue.

How Jane Thinks

Jane automatically picks the right brain for each message:

Message typeModelSpeed
Greetings ("hey", "thanks")Haiku (fast)~5s
Simple questions ("what time is it")Haiku~5s
Medium tasks ("fix this bug")Sonnet~10-20s
Complex work ("build this feature")Opus (powerful)~15-60s

The model label appears at the bottom-right of each response so you can see which brain handled it.

White text = Jane's thoughts. While working on complex tasks, you'll see Jane's intermediate thinking (reading files, running searches, editing code) in small white text. These collapse into a summary after the response finishes.

Features

  • File upload: Drag-and-drop or use the attach button for PDFs, images, documents
  • Voice: Click "Play audio" on any response to hear it read aloud. Toggle TTS in settings.
  • LaTeX: Jane renders math: $E=mc^2$ for inline, $$\sum_{i=1}^n$$ for display
  • Code: Syntax-highlighted code blocks with copy button
  • Daily Briefing: Personalized news with brief/full audio summaries
  • Memory: Jane remembers across sessions. She'll reference past conversations naturally.

Troubleshooting

  • "Jane is offline" — Usually recovers in 30 seconds. Try a hard refresh (Ctrl+Shift+R).
  • Slow first response after restart — The AI model is warming up. Subsequent messages will be faster.
  • No essences showing — Click the dropdown again. Data pre-fetches on page load.
  • Response cuts off — Jane may have hit a timeout. Send "continue" to pick up where she left off.
