Install Vessence

Get up and running in minutes. Vessence runs entirely on your machine via Docker — your data never leaves your computer.

Prerequisites

  • Docker Desktop (Windows/Mac) or Docker Engine + Compose plugin (Linux) — required to run the containers
  • A Google API key (free tier) — required for Gemini brain
  • Optional: Anthropic API key (Claude) or OpenAI API key for additional brain options

Windows Installation

  1. Install Docker Desktop
    Download and install Docker Desktop for Windows. Enable WSL2 when prompted during setup.
  2. Download the installer
    Click the download button below to get the Vessence Windows installer package.
  3. Extract the zip
    Right-click the downloaded vessence-windows-installer-v0.0.12.zip and select "Extract All..."
  4. Unblock the installer
    Right-click Install Vessence.bat → Properties → check Unblock → click OK.
  5. Run the installer
    Double-click Install Vessence.bat. The installer will load Docker images and set up everything automatically.
Troubleshooting: If you see "Docker not found", restart your computer after installing Docker Desktop and try again.

macOS Installation

  1. Install Docker Desktop
    Download and install Docker Desktop for Mac. Apple Silicon (M1/M2/M3/M4) is fully supported.
  2. Download the installer
    Click the download button below to get the Vessence Mac installer package.
  3. Extract and run
    Extract the zip, then open Terminal and run:
    chmod +x install-mac.command && ./install-mac.command
  4. Alternative: double-click
    You can also double-click install-mac.command in Finder. If macOS blocks it, right-click the file and select Open.
Apple Silicon: Both Intel and Apple Silicon Macs are supported. Docker will use the correct images automatically.
Download for Mac

Linux Installation

  1. Install Docker Engine
    Run in your terminal:
    curl -fsSL https://get.docker.com | sh
  2. Install the Compose plugin
    sudo apt install docker-compose-plugin
  3. Download the installer
    Click the download button below to get the Vessence Linux installer package.
  4. Extract and run
    Extract the zip and run the installer:
    bash install-linux.sh

What happens during install

Docker images load ~210 MB — the core Vessence containers
ChromaDB pulls ~350 MB — vector memory database from Docker Hub
Brain CLI installs ~200–400 MB — your chosen AI brain downloads on first boot
Onboarding wizard Guides you through API key setup and personal preferences

After install

Jane Chat http://localhost:8081 Talk to your AI persona
Onboarding / Settings http://localhost:3000 Configure API keys and preferences
100% Local Your data stays on your machine Nothing is sent to Vessence servers
Vessence — your AI personas, private and portable. Free for local use.
