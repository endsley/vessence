# Project: Local Google Photos Sync & Intelligence

## Description
This project aims to integrate with Google Photos API for downloading photos, implement local vision/facial recognition, and develop an Android sync app.

## Functionalities
-   **Phase 1 (Collector):** Google Photos API integration (OAuth2, list_library, download_photo, sync_all with throttling).
-   **Phase 2 (Analyst):** Local facial recognition (DeepFace/dlib, 128-d encodings, identity mapping for User/Wife/Daughter, auto-sorting to family folders, and semantic memory tagging).
-   **Phase 3 (Pipeline):** Android sync app pushing new mobile photos directly to Amber's local FastAPI receiver.

## Use Cases
-   Semantically organizing a local photo library.
-   Enabling natural language retrieval of photos based on recognized individuals or content.
-   Automating photo synchronization from mobile devices.
