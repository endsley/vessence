# Project: Multi-Modal "See & Do" Automation

## Description
This project aims to integrate vision capabilities ("Sight") with interaction capabilities ("Motor Brain") to enable autonomous web tasks.

## Functionalities
-   **Search Brain Integration:** Connecting search functionality to guide actions.
-   **Motor Brain Integration:** Enabling autonomous navigation and interaction with web pages.
-   **Complex Task Execution:** Handling tasks like flight booking and research through combined intelligence.
-   **Amber's Autonomous Navigation:** Allowing Amber to navigate websites autonomously.
-   **Visual Summaries:** Providing visual summaries of web interactions via screenshots.

## Use Cases
-   Automated web research and data gathering.
-   Performing complex online transactions (e.g., booking flights).
-   Amber acting as an autonomous agent for specific web-based tasks.
